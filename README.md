# APO Website

Flask site, attempt to make a new APO site

## Setup

Clone the repository and enter it

```
git clone https://github.com/RafaelCenzano/apo-site.git
cd apo-site
```

### Requirements

Use pip to install needed libraries

```
make
```

or

```
pip install -r requirements.txt
```

### Run 

```
make run
```

or

```
gunicorn run:app
```
